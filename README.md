# Plagiarism Checker

A web-based plagiarism detection system built using Python and Flask.

## Features
- Upload Word documents (.docx)
- Extracts text from uploaded documents
- Generates random phrases for plagiarism detection
- Performs best-effort web search
- Calculates plagiarism percentage using TF-IDF and cosine similarity
- Clean and user-friendly UI

## Tech Stack
- Python
- Flask
- HTML, CSS
- BeautifulSoup
- scikit-learn
